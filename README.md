# Yesodbook中文翻译计划

[Yesod]是一个用[Haskell]写的web框架。学习Haskell和Yesod，建议尽量阅读原文。

- [Yesodbook源文件]
- [Yesodbook在线版]
- [Learn You a Haskell for Great Good!]
- [Real World Haskell]

翻译这本书完全是为了督促自己深入学习Yesod。我读过的中文技术书籍极为有限，词汇量也非常小，慎读！
